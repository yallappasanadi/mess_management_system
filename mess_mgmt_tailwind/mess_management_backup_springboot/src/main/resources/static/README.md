Mess Management Frontend (Tailwind)
This frontend-only project uses HTML, Tailwind CDN, and simple JavaScript to simulate:
- Home page with overview (total/present/absent)
- Students page listing entry/exit times
- Menu page showing breakfast/lunch/dinner
- Signin and Signup pages (frontend only)

Files:
- index.html (home)
- students.html
- menu.html
- signin.html
- signup.html
- styles.css
- app.js

SQL file reference (local): /mnt/data/mess_management_system.sql

How to use:
1. Extract the zip and open index.html in a browser.
2. Clicking overview cards opens separate windows (students/menu).
3. All data is mock data inside app.js. To connect to backend replace mocks with fetch() calls.
